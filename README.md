# 🚀 DeadlineAI

DeadlineAI is an AI-powered productivity assistant that helps students and professionals organize tasks, meet deadlines, and plan their day intelligently. It combines Google Gemini AI with Firebase and Google Cloud to create personalized schedules, task recommendations, and an interactive AI assistant.

## 📌 How It Works

1. Sign in using your Google account.
2. Add tasks using natural language.
3. Gemini AI analyzes deadlines and priorities.
4. Tasks are automatically organized into a smart schedule.
5. The AI Assistant can create, edit, and manage tasks through chat.
6. Users receive productivity insights based on their workload.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/sampadanayak504/DeadlineAI.git
```

Navigate into the project

```bash
cd DeadlineAI
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
